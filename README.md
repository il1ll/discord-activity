## API

this project uses 2 APIs

### 1 — Lanyard API
link:  
`https://api.lanyard.rest/v1/users/{your id}`

this api returns all account info:
- avatar  
- decoration  
- nameplate  
- tag  
- badges  
- presence (desktop - mobile - web - console)  
- online status (online - idle - dnd - offline)  
- activity status  
- custom activity + custom emoji  
- spotify activity  

### 2 — dstn CDN
link:  
`https://dcdn.dstn.to/profile/{your id}`

this api returns:
- banner image  
- banner color  
- extra info (not needed)
